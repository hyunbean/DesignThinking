# [E1] 인터뷰 대상자 선정 및 섭외 전략

**입력**: SMART 목표(4주 내 대학생 소비·저축 페인포인트 검증 및 컨셉 도출) + 가용 자원(팀원 4명, 4주)

---

[인터뷰 대상자 선정 및 섭외 전략]

## 1. 인터뷰 대상자 선정 기준

1. **소득 구조 적합성**: 월 40~80만원의 용돈/알바 소득으로 생활하는 대학생 — 목표가 정의한 "월 소득 60만원 내외"와 직접 연결
2. **저축 시도 경험**: 저축을 시도했으나 지속하지 못한 경험이 있는 사람 (실패 요인이 곧 페인포인트)
3. **관리 도구 사용 경험**: 가계부 앱·뱅킹 앱의 저축 기능(세이프박스류)을 써봤거나 중도 포기한 사람 — 기존 솔루션의 한계를 증언할 수 있음
4. **소비 성향 다양성**: 충동소비형·계획소비형을 모두 포함해 특정 성향에 편향되지 않도록 함

## 2. 잠재적 인터뷰 대상자를 찾고 접근하기 위한 방법과 전략

- **교내 채널**: 학과 단톡방·에브리타임에 짧은 사전 설문(구글폼)을 올려 소득 구조·저축 경험 기준으로 스크리닝 후 인터뷰 요청
- **지인 스노볼링**: 팀원 4명이 각자 조건에 맞는 지인 1~2명씩 섭외, 그 지인의 소개로 확장
- **온라인 커뮤니티**: 대학생 재테크·앱테크 커뮤니티에서 가계부 앱 사용 후기를 남긴 사용자에게 접근 (관리 도구 중도 포기 경험자 확보)
- **보상 설계**: 인터뷰 30~40분당 커피 기프티콘 제공 — 예산은 팀 회비 내에서 인당 5천원 수준

## 3. 인터뷰 과정을 조정하기 위한 실질적 단계

1. 1주차: 스크리닝 설문 배포 → 대상자 리스트 작성 (유형별 목표 인원 충족 확인)
2. 1~2주차: 접촉·일정 조율, 인터뷰 가이드라인(질문지) 확정
3. 2~3주차: 팀원 4명이 2인 1조(진행 1명 + 기록 1명)로 인터뷰 수행
4. 매 인터뷰 직후: 원본 발언을 THOUGHTS(해석)와 분리해 기록 — Define 단계 입력 품질 확보
5. 3주차 말: 기록 취합 → 클러스터링([D1])으로 진행

## 4. 추천 인터뷰 대상자 유형 breakdown

| 유형명 | 인원수 | 선정 이유 |
|---|---|---|
| 현재 대학생 소비자 (용돈/알바 소득) | 3명 | 목표 세그먼트 본체. 소비 패턴·저축 실패 경험의 1차 데이터 소스 |
| 저축 성공 경험자 (목표 달성 경험) | 1명 | 성공 요인을 역으로 추출해 실패군과 대조 |
| 가계부 앱 사용 중단자 | 2명 | 기존 솔루션(수동 기록형)의 이탈 요인 검증 |
| 금융 서비스 기획·운영 경험자 | 1명 | 시장·규제 관점의 현실성 확인 (업계 전문가 역할) |

**합계 7명** — 팀원 4명이 4주(실질 인터뷰 기간 2주) 안에 수행 가능한 규모.
